# 내기 리더보드

친구들 종목 내기 등락률 리더보드. GitHub Pages(정적) + GitHub Actions(자동 갱신).

## 구조
- `index.html` — 비밀번호 입력 → 브라우저에서 복호화 → 렌더 → 60초마다 자동 새로고침
- `build.mjs` — Yahoo Finance에서 5종목 시세 조회 → 등락률 계산 → AES-256-GCM 암호화 → `data.enc`
- `data.enc` — 암호화된 데이터(비밀번호 없이는 못 봄). Actions가 약 10분마다 갱신
- `.github/workflows/update.yml` — 10분 주기 크론 + 수동 실행

## 설정
- 비밀번호는 저장소 Secret `LEADERBOARD_PW`로 관리. 로컬 생성 시 `LEADERBOARD_PW=<비번> node build.mjs`.
- 보안 메모: public repo이지만 데이터는 암호화되어 올라가므로 비밀번호 없이는 내용을 볼 수 없다. 다만 이는 정적 호스팅상의 경량 보호이며, 비밀번호가 유출되면 데이터도 노출된다.
